1Resumen

Por medio de la siguiente pantalla es posible realizar el registro de un proyecto curricular (Crea un directorio con el nombre asigando al proyecto curricular) enlazandolo con la facultad correspondiente en la cual se imparte.

2Guía rápida

1. Seleccione la Facultad a la que pertenece el proyecto curricular a agregar
2. Ingrese el nombre del Proyecto curricular.
